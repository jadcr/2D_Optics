0.01 - Реализовано моделирование 2D трассировки лучей, которое включает:
- Источник лучей: параллельный пучок (начало, направление, кол-во лучей, шаг), точечный источник (начало, направление, кол-во лучей, угловое распределение)
- Список элементов (для всех элементов реализован флаг отражения/преломления):
    - Плоская граница раздела/полуплоскость - отрезок (показатель преломления до и после границы, точки начала и конца отрезка)
    - Стекло в форме круга (центр, радиус, показатель преломления внутри и снаружи)
    - Детектор - отрезок (сохранение точек прихода лучей, точки начала и конца отрезка)
- Декартову систему координат
Доступный функционал:
- Возможность как менять положение источника и элементов вручную, так и линейно двигать их левой кнопкой мыши с перетрассировкой в реальном времени
- Удалять выбранный элемент, менять имя элемента
- Включать, выключать трассировку
- Удалять все объекты со сцены
